🗑️ Sorting Master
Sorting Master คือเกมแนว Casual อาเขตที่เขียนขึ้นด้วยภาษา Python และไลบรารี Pygame ผู้เล่นจะต้องรับบทเป็นพนักงานคัดแยกขยะ โดยต้องลากขยะที่ไหลลงมาจากสายพาน ไปทิ้งให้ตรงตามประเภทของถังขยะเพื่อสะสมคะแนนให้ได้มากที่สุดก่อนที่พลังชีวิตจะหมดลง!

✨ ฟีเจอร์หลัก (Features)
Drag & Drop Mechanics: ระบบลากและปล่อยขยะที่ควบคุมง่าย ลื่นไหล

4 ประเภทขยะ: คัดแยกขยะให้ถูกต้อง ได้แก่ ขยะรีไซเคิล (Recycle), ขยะเปียก (Wet), ขยะทั่วไป (General) และ ขยะอันตราย (Hazardous)

ระบบคะแนนและพลังชีวิต (Score & Lives): ทิ้งถูกถังได้คะแนนสะสม หากทิ้งผิดหรือปล่อยขยะร่วงหล่นสายพานจะเสียพลังชีวิต

ระบบ Pause Game: สามารถกดหยุดพักเกมชั่วคราวได้โดยที่เวลาของเกมไม่เดินต่อ

กระดานผู้นำ (Top 10 Leaderboard): บันทึกคะแนนสูงสุด 10 อันดับแรกพร้อมเวลาที่ใช้เล่นโดยอัตโนมัติ (ซ่อนกระดานหากยังไม่มีประวัติการเล่น)

🛠️ ความต้องการของระบบ (Prerequisites)
ก่อนที่คุณจะรันเกมนี้ได้ ต้องติดตั้งโปรแกรมดังต่อไปนี้ในเครื่อง:

Python 3.x

ไลบรารี Pygame

🚀 วิธีติดตั้งและรันเกม (Installation & Run)
ติดตั้งไลบรารี Pygame โดยเปิด Terminal หรือ Command Prompt แล้วพิมพ์คำสั่ง:

Bash
pip install pygame
ดาวน์โหลดไฟล์โค้ดของเกมและไฟล์รูปภาพทั้งหมด (Assets) ไปไว้ในโฟลเดอร์เดียวกัน

เปิด Terminal นำทางไปยังโฟลเดอร์ที่เก็บเกมไว้ แล้วรันคำสั่ง:

Bash
python alpha0.5.6.py


🎮 วิธีการเล่น (How to Play)
เป้าหมาย: ทำคะแนนให้ได้มากที่สุดและอยู่รอดให้นานที่สุด

การควบคุม: ใช้ เมาส์คลิกซ้ายค้าง ที่ชิ้นขยะเพื่อหยิบขึ้นมา แล้วลากไป ปล่อย (Drop) ลงในถังขยะที่ถูกต้อง

กฎกติกา:

✅ ทิ้งลงถังถูกต้อง: +10 คะแนน

❌ ทิ้งผิดถัง: เสีย 1 พลังชีวิต (ไม่หักคะแนน)

⚠️ ปล่อยขยะร่วงตกขอบจอ: เสีย 1 พลังชีวิต

🛑 เกมจะจบลงเมื่อพลังชีวิต (Lives) เหลือ 0

สามารถกดปุ่ม PAUSE ที่มุมขวาบนเพื่อหยุดเกมชั่วคราวได้
